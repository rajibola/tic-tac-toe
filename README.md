## Getting Started

- Fork or Clone the repo, then set it up:

```
$ cd tic-tac-toc
$ yarn install
```

### Run on Android

```
$ yarn run android
```

### Run on iOS

```
$ cd ios && pod install
$ cd .. && yarn run ios
```
